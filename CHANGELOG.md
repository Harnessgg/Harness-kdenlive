# Changelog

## 0.4.0

- Added end-to-end project/timeline/asset/track/render command surface.
- Added doctor diagnostics with optional breakage reporting.
- Added bridge soak stability command.
- Added plan-edit preview and dry-run support for timeline/asset import operations.
- Added undo/redo command surface backed by snapshot history.
- Added effect and transition list/apply/update/remove actions and CLI commands.
- Added render presets (`h264`, `hevc`, `prores`), zone render support, and render status lookup.
- Added response schema validation tests and expanded bridge integration coverage.
- Added regression fixture set (`small`, `medium`, `complex`) and tests.
